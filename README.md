# Ocean Fleet App
A console-based Java application for tracking vessel performance and identifying high-performing cargo ships.

- **UC1: Vessel Model** - Defines the core `Vessel` class with properties like ID, name, speed, and type.
- **UC2: List Storage** - Implements `VesselUtil` to store vessel objects in a dynamic `ArrayList`.
- **UC3: Search by ID** - Enables retrieval of specific vessel details using a case-sensitive ID search.
- **UC4: Performance Analytics** - Identifies vessels with the highest average speed across the fleet.
- **UC5: Console UI** - Provides a text-based interface for user input and data presentation.
- **UC6: OOPS Refactoring** - Enhances maintainability by applying encapsulation and separation of concerns.
